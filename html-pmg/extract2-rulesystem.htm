Object-oriented Rule System

The rule system enables you to define filter rules by user, domains, time frame, content type and resulting action. It is very flexible and easy to use.

Options range from simple spam and virus filter setups to sophisticated, highly customized setups blocking certain types of mails and generating notifications.

Its flexibility enables easy integration into your existing network architecture.

Who-objects
The action of a rule will only be triggered if the e-mail contains the defined who-object.

What-objects
A what-object defines what will happen to an e-mail with a specific content.

When-objects
Set a time frame. The rule will only work in the specified time frame.

Action-objects
Basic actions: quarantine, block, accept, add disclaimer, notify ...
